（自主制作） Oshiro Naho Portfolio

制作日：2024年10月

https://naho-oshiro.com

  • ターゲット

    Web制作会社の採用担当者の方々

  • 目的

    就職活動を行うにあたり、自身の経歴や強み、またこれまでの制作物や思考を伝えるため

  • 概要

    TOPページをシンプルに作成し、大体の概要を知っていただいた後、Aboutページでスキルや経歴、強みを詳細に書くことで説得力のある実体験を元にしたプロフィールの作成に努めました。 制作物についても項目を分けて分かりやすく制作の裏側が分かるように構成しました。

  • デザイン

    全体的なサイトのデザイン、色使いは「オレンジや黄色のエネルギッシュなカラー」を使うことで自分らしさを表現しながらも、フォントや文章の書き方で誠実さも感じられるデザインにしました。 テキスト量が多くなってしまうAboutページは、ブロックごとに色分けを行ったり、構成を変えることで飽きのこない工夫を行いました。

  • 制作範囲・期間

    企画、ワイヤーフレーム：5日 デザイン：2週間 コーディング：3週間

  • 使用ツール

    Illustrator / Photoshop / XD / Visual Studio Code
