（自主制作） 野菜の通販サイト（架空）

制作日：2024年12月

  • ターゲット

    食へのこだわりがある、宅配での購入を検討している20代後半～60代の男女

  • 目的

    産地や栽培方法にこだわりのある層に定期的に購入してもらう

  • 概要

    新鮮な野菜を安心して購入できるように、商品に農家さんの情報をセットで載せています。訪問者が容易に知りたい情報に届くように、検索機能だけでなく、サイドメニューにて項目を設けたり、ヘッダー下のナビゲーションを細かく設定しています。商品の情報を見た後に、実際の購入者の声を載せることで購買意欲を掻き立て、最後に定期便コースの詳細を載せてCTAボタンに誘導しています。

  • デザイン

    全体的にナチュラルで、親しみのあるデザインを意識しました。緑黄色野菜をイメージして、ベースをベージュにしタイトルやサイドメニューで緑のサブカラーを使い、CTAボタンをオレンジで強調して、色ごとに役割を分けることで全体の統一性と見やすさを考えて作成しました。

  • 制作範囲・期間

    企画、ワイヤーフレーム：３日間
    デザイン：１週間
    コーディング：3週間

  • 使用ツール

    Illustrator / Photoshop / XD / Visual Studio Code
